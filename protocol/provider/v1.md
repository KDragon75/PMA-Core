# Provider protocol v1

The provider uses newline-delimited JSON-RPC 2.0 over stdio. Required methods are `provider.initialize`, `provider.describe_runtime`, `provider.get_capabilities`, `provider.health`, `provider.cancel`, `provider.shutdown`, `embedding.embed_documents`, `embedding.embed_query`, and `generation.generate_structured`.

Initialization records all effective model, tokenizer, pooling, normalization, prefix, truncation, device, dtype, artifact, package-lock, and runtime settings. Models load lazily. Structured generation extracts one JSON value, validates it, and retries once with validation feedback. Stdout contains protocol messages only.
